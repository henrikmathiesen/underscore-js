What is underscorejs?

This Javascript library provides around 80 functions that helps with every day Javascript tasks, including templating. It requires no dependencies. It works in the browser and also in NodeJS.

Underscorejs checks if there is a native Javascript implementation of a function and will defer to that in that case.

Underscore has two styles of coding; functional and object style, which to use is a matter of preference.

Functional

var m = _.map(names, function(name) { return "Hello " + name; });

Object style

var m = _(names).map(function() { return "Hello " + name; });
