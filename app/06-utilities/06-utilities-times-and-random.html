UnderscoreJs

- Times and Random

http://underscorejs.org/#times times_.times(n, iteratee, [context]) Invokes the given iteratee function n times. Each invocation of iteratee is called with an index argument. Produces an array of the returned values. Note: this example uses the object-oriented syntax.

http://underscorejs.org/#random random_.random(min, max) Returns a random integer between min and max, inclusive. If you only pass one argument, it will return a number between 0 and that number.
